# Auto Capture Game Script Real Time

This project captures game dialogue in real-time, extracts text using OCR, translates it via OpenRouter API, and displays it on an overlay.

## Setup

1. Install `uv` if you haven't already.
2. Run `uv sync` to install dependencies.
3. Copy `.env.example` to `.env` and configure your API key.
4. Run `uv run main.py`.
